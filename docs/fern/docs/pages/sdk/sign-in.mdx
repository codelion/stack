# SignIn

A component that renders a sign-in page using the AuthPage component.

## Props

- `fullPage` (optional): `boolean` - Determines whether to display the sign-in page in full-page mode. Default is `false`.

## Example

```tsx
import { SignIn } from '@stackframe/stack';

// Default usage
<SignIn />

// Full-page mode
<SignIn fullPage={true} />
```
