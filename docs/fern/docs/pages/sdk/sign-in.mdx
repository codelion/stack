# SignIn

Renders a sign-in authentication page. By default, it renders a partial page unless specified otherwise.

## Parameters

- `fullPage` (optional): `boolean` - Determines whether to render the full page. Defaults to `false`.

## Example

```tsx
import { SignIn } from '@stackframe/stack';

// Render sign-in page as a partial (default behavior)
<SignIn />

// Render sign-in page as a full page
<SignIn fullPage={true} />
```
