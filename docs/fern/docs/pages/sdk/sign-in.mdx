# SignIn

Renders a sign-in page. This component can be rendered as a full page or as part of another page.

## Parameters

- `fullPage` (optional): `boolean` - Determines if the sign-in page should be rendered as a full page. Defaults to `false`.

## Example

```tsx
import { SignIn } from '@stackframe/stack';

// Render sign-in as part of another page
<SignIn />

// Render sign-in as a full page
<SignIn fullPage={true} />
```
