# SignIn

Renders the sign-in authentication page.

## Parameters

- `fullPage`: `boolean` (optional) - If true, the authentication page will be rendered in full-page mode. Default is false.

## Returns

`JSX.Element` - A rendered sign-in authentication page component.

## Example

```typescript
import { SignIn } from 'your-sdk-name';

const App = () => {
  return <SignIn fullPage={true} />;
};
```
