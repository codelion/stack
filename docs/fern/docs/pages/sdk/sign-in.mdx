# SignIn

A React component that renders a sign-in page. It can be displayed as a full page or embedded within another component.

## Parameters

- `fullPage` (optional): `boolean` - Determines whether the sign-in page should be rendered as a full page. Default value is `false`.

## Returns

Returns a React component that displays a sign-in page.

## Example

```tsx
import { SignIn } from '@stackframe/stack';

// Render as an embedded component
function App() {
  return (
    <div>
      <h1>Welcome to our app</h1>
      <SignIn />
    </div>
  );
}

// Render as a full page
function FullPageSignIn() {
  return <SignIn fullPage={true} />;
}
