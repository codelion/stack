# SignIn

Renders a sign-in authentication page with an optional full-page layout.

## Parameters

- `fullPage`: `boolean` (optional) - Determines whether to display the sign-in page in full-screen mode. Defaults to `false`.

## Returns

`JSX.Element` - A rendered sign-in authentication page component.

## Example

```typescript
import { SignIn } from '@stackframe/stack';

function App() {
  return <SignIn />;
}
```
