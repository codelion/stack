# LINK_COLORS

An object containing color codes for different link themes. Use this to apply consistent colors to your links based on the selected theme.

## Properties

- `dark`: `#fff` - The color code for links when using the dark theme.
- `light`: `#000` - The color code for links when using the light theme.

## Example

```tsx
import { LINK_COLORS } from '@stackframe/stack';

const darkLinkColor = LINK_COLORS.dark; // '#fff'
const lightLinkColor = LINK_COLORS.light; // '#000'
```
