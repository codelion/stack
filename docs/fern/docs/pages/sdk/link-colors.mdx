# LINK_COLORS

A constant object that defines color values for links in dark and light themes.

## Properties

- `dark`: `'#fff'` - The color value for links in dark theme.
- `light`: `'#000'` - The color value for links in light theme.

## Example

```tsx
import { LINK_COLORS } from '@stackframe/stack';

// Using the link colors
const darkThemeColor = LINK_COLORS.dark;
// darkThemeColor is '#fff'

const lightThemeColor = LINK_COLORS.light;
// lightThemeColor is '#000'
```// darkThemeColor is '#fff'

const lightThemeColor = LINK_COLORS.light;

```
