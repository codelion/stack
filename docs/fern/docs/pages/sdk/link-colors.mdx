# LINK_COLORS

A constant object that defines link colors for different themes.

## Properties

- `dark`: `'#fff'` - The color for links in dark theme.
- `light`: `'#000'` - The color for links in light theme.

## Example

```tsx
import { LINK_COLORS } from '@stackframe/stack';

const darkThemeLinkColor = LINK_COLORS.dark;
// darkThemeLinkColor is '#fff'

const lightThemeLinkColor = LINK_COLORS.light;
// lightThemeLinkColor is '#000'
```
