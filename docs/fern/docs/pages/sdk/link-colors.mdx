# LINK_COLORS

An object that defines color values for different link styles.

## Properties

- `dark`: `string` - The color value for dark themed links.
- `light`: `string` - The color value for light themed links.

## Example

```typescript
import { LINK_COLORS } from '@stackframe/stack';

console.log(LINK_COLORS.dark); // Output: #fff
console.log(LINK_COLORS.light); // Output: #000
```
