# LINK_COLORS

An object containing predefined link colors for a dark and light theme.

## Properties

- `dark`: `string` - The color for links in dark mode, set to `#fff`.
- `light`: `string` - The color for links in light mode, set to `#000`.

## Example

```typescript
import { LINK_COLORS } from 'your-sdk-name';

const darkLinkColor = LINK_COLORS.dark;
const lightLinkColor = LINK_COLORS.light;
console.log(darkLinkColor); // Output: #fff
console.log(lightLinkColor); // Output: #000
```
