# SeparatorWithText

A component that displays a separator with text centered between two separator lines.

## Parameters

- `text`: `string` - The text to display between the separator lines.

## Example

```tsx
import { SeparatorWithText } from '@stackframe/stack';

const MyComponent = () => (
  <div>
    <SeparatorWithText text='Section Title' />
  </div>
);
```
