# SeparatorWithText

Renders a horizontal separator with a text label centered between two separator lines. The separator helps to visually separate content sections on a page.

## Parameters

- `text`: `string` - The text to display in the center of the separator.

## Example

```tsx
import { SeparatorWithText } from '@stackframe/stack';

const MyComponent = () => {
  return (
    <div>
      <SeparatorWithText text='Section Title' />
    </div>
  );
};
```
