# setOrDeleteCookie

Sets a cookie with a specified name and value, or deletes the cookie if the value is null.

## Parameters

- `name`: `string` - The name of the cookie to set or delete.
- `value`: `string | null` - The value of the cookie. If `null`, the cookie will be deleted.
- `options`: `SetCookieOptions` (optional) - Options to configure the cookie when setting it.

## Example

```typescript
import { setOrDeleteCookie } from '@stackframe/stack';

// Set a cookie
setOrDeleteCookie('userToken', 'abc123');

// Delete a cookie
setOrDeleteCookie('userToken', null);
```
