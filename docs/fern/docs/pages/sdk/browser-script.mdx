# BrowserScript

The `BrowserScript` function generates a browser-executable script with an optional nonce attribute. It utilizes the `SsrScript` component to embed the script in Server-Side Rendering (SSR) and assigns the provided nonce if available.

## Parameters

- `props`: `Object` - The properties object.
  - `nonce` (optional): `string` - A nonce value for the script tag, used for Content Security Policy (CSP).

## Example

```tsx
import { BrowserScript } from '@stackframe/stack';

// Without nonce
const scriptElementWithoutNonce = <BrowserScript />;

// With nonce
const scriptElementWithNonce = <BrowserScript nonce='your-nonce-value' />;
```
