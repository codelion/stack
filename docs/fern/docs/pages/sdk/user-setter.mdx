# UserSetter

Sets the current user in the stack application using a promise.

## Parameters

- `props`: `object` - An object containing the following property:
  - `userJsonPromise`: `Promise<CurrentUserCrud['Client']['Read'] | null>` - A promise that resolves to the current user data or `null`.

## Returns

`null` - This function does not return any value.

## Example

```typescript
import { UserSetter } from 'your-sdk-name';

const userJsonPromise = fetchUserJson(); // Assume this is a function that returns a promise

<UserSetter userJsonPromise={userJsonPromise} />;
```
