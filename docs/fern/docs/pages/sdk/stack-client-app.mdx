# StackClientApp

`StackClientApp` is a constructor for creating instances of the Stack client application.

## Usage

```tsx
import { StackClientApp } from '@stackframe/stack';

// Create an instance of StackClientApp
const app = new StackClientApp();
```

## Example

```tsx
import { StackClientApp } from '@stackframe/stack';

// Initialize the Stack client application
const app = new StackClientApp();

// Use the app instance as needed
// app.someMethod();
```
