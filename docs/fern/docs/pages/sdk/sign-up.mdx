# SignUp

Renders a sign-up page component. Can be displayed as a full page or embedded within other content.

## Props

- `fullPage` (optional): `boolean` - Determines whether to render the sign-up page in full-page mode. Defaults to `false`.

## Example

```tsx
import { SignUp } from '@stackframe/stack';

// Default usage (embedded)
<SignUp />

// Full-page mode
<SignUp fullPage={true} />
```
