# SignUp

Renders a sign-up authentication page. This component can be displayed as a full page or within a container.

## Parameters

- `fullPage` (optional): `boolean` - Determines if the sign-up page should be displayed as a full page. Defaults to `false`.

## Example

```tsx
import { SignUp } from '@stackframe/stack';

// Render sign-up page within a container
<SignUp />

// Render sign-up page as a full page
<SignUp fullPage={true} />
```
