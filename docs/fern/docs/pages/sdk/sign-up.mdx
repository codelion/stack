# SignUp

Renders a sign-up authentication page with an option to display it in full page mode.

## Parameters

- `fullPage`: `boolean` (optional) - A flag indicating whether to display the authentication page in full-page mode. Defaults to `false`.

## Example

```typescript
import { SignUp } from 'your-sdk-name';

const App = () => {
  return <SignUp fullPage={true} />;
};
```
