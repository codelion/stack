# SignUp

A component that renders a sign-up page or form. It wraps the AuthPage component with pre-configured settings for sign-up functionality.

## Parameters

- `fullPage` (optional): `boolean` - Determines whether to render the sign-up form as a full page. Default is `false`.

## Example

```tsx
import { SignUp } from '@stackframe/stack';

// Default usage (embedded form)
function MyComponent() {
  return <SignUp />;
}

// Full page sign-up
function FullPageSignUp() {
  return <SignUp fullPage={true} />;
}
