# SignUp

Renders a sign-up authentication page.

## Parameters

- `fullPage`: `boolean` (optional) - Specifies whether the sign-up page should occupy the full viewport height. Defaults to `false`.

## Returns

`JSX.Element` - A rendered sign-up authentication page component.

## Example

```typescript
import { SignUp } from '@stackframe/stack';

const App = () => {
  return <SignUp />;
};
```
