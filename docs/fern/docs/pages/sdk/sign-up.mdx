# SignUp

Renders a Sign-Up authentication page. Returns a React component representing the Sign-Up page.

## Parameters

- `fullPage` (optional): `boolean` - Determines if the Sign-Up page should be rendered as a full page. Defaults to `false`.

## Example

```tsx
import { SignUp } from '@stackframe/stack';

// Default usage
<SignUp />

// Full page usage
<SignUp fullPage={true} />
```
