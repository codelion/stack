# getVerifierAndState

Retrieves the code verifier and state from cookies.

## Returns

`{ codeVerifier: string, state: string }` - An object containing the code verifier and state retrieved from cookies.

## Example

```typescript
import { getVerifierAndState } from '@stackframe/stack';

const { codeVerifier, state } = getVerifierAndState();
console.log(codeVerifier, state);
```
