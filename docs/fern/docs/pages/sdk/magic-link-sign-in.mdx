# MagicLinkSignIn

A component that allows users to sign in using a magic link sent to their email.

## Returns

This component does not return any value. It renders a form for email input and handles the submission to send a magic link.

## Example

```typescript
import { MagicLinkSignIn } from '@stackframe/stack';

function App() {
  return (
    <div>
      <h1>Sign In</h1>
      <MagicLinkSignIn />
    </div>
  );
}
```
