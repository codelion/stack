# MagicLinkSignIn

A component that allows users to sign in using a magic link sent to their email.

## Parameters

This component does not accept any parameters.

## Returns

This component does not return a value but renders a form.

## Usage

```typescript
import { MagicLinkSignIn } from 'your-sdk-name';

const App = () => (
  <div>
    <h1>Sign In</h1>
    <MagicLinkSignIn />
  </div>
);

export default App;
```
