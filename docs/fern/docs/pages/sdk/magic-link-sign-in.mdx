# MagicLinkSignIn

A React component for magic link sign-in functionality. It handles form submission and displays the sign-in form.

## Parameters

This component doesn't accept any parameters.

## Example

```tsx
import { MagicLinkSignIn } from '@stackframe/stack';

function SignInPage() {
  return (
    <div>
      <h1>Sign In</h1>
      <MagicLinkSignIn />
    </div>
  );
}
```
