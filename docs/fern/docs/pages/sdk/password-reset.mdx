# PasswordReset

Renders the appropriate password reset component based on the given `searchParams`. This includes handling invalid, expired, or already used reset links.

## Parameters

- `searchParams`: `Record<string, string>` - An object containing search parameters, expected to have a `code` key for the password reset code.
- `fullPage` (optional): `boolean` - Determines if the message cards should be displayed in full page mode. Defaults to `false`.

## Example

```tsx
import { PasswordReset } from '@stackframe/stack';

const searchParams = { code: 'someResetCode' };

// Basic usage
<PasswordReset searchParams={searchParams} />;

// Full page mode
<PasswordReset searchParams={searchParams} fullPage={true} />;
```
