# PasswordReset

Displays the password reset interface based on the provided parameters.

## Parameters

- `searchParams`: `Record<string, string>` - An object containing search parameters, including the password reset code.
- `fullPage`: `boolean` (optional) - Determines whether to render the reset UI as a full page. Default is `false`.

## Returns

`JSX.Element` - Renders a specific message or the password reset form based on the verification state of the code.

## Example

```tsx
import { PasswordReset } from 'your-sdk-name';

const SearchParams = { code: 'your-reset-code' };

<PasswordReset searchParams={SearchParams} fullPage={true} />
```
