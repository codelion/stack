# PasswordReset

The `PasswordReset` function component handles the password reset process by validating the reset link and displaying appropriate messages or the password reset form.

## Parameters

- `searchParams`: `Record<string, string>` - An object containing query parameters from the URL. Must include a `code` parameter for the password reset.
- `fullPage` (optional): `boolean` - A flag indicating whether the component should be displayed as a full page. Defaults to `false`.

## Example

```tsx
import { PasswordReset } from '@stackframe/stack';

const searchParams = { code: 'exampleCode' };

// Basic usage
<PasswordReset searchParams={searchParams} />;

// Full page usage
<PasswordReset searchParams={searchParams} fullPage={true} />;
```
