# PasswordReset

Renders the password reset component based on the provided search parameters.

## Parameters

- `searchParams`: `Record<string, string>` - An object containing search parameters for password reset, including a verification code.
- `fullPage`: `boolean` (optional) - A flag to indicate if the password reset should be displayed as a full page.

## Returns

`JSX.Element` - A React element representing the password reset form or an error message if the link is invalid, expired, or already used.

## Example

```typescript
import { PasswordReset } from '@stackframe/stack';

const searchParams = { code: 'your-verification-code' };

<PasswordReset searchParams={searchParams} fullPage={true} />;
```
