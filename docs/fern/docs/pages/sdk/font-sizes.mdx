# FONT_SIZES

A constant object that defines font sizes for different labels. Each key represents a label, and the value is the corresponding font size in `rem` units.

## Properties

- `xs`: `0.75rem` - Extra small font size.
- `sm`: `0.875rem` - Small font size.
- `md`: `1rem` - Medium font size.
- `lg`: `1.125rem` - Large font size.
- `xl`: `1.25rem` - Extra large font size.

## Example

```tsx
import { FONT_SIZES } from '@stackframe/stack';

const smallFontSize = FONT_SIZES.sm;
// smallFontSize is '0.875rem'

const largeFontSize = FONT_SIZES.lg;
// largeFontSize is '1.125rem'
```
