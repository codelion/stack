# FONT_SIZES

A constant object that defines standard font sizes used in the application.

## Properties

- `xs`: `string` - Extra small font size (0.75rem).
- `sm`: `string` - Small font size (0.875rem).
- `md`: `string` - Medium font size (1rem).
- `lg`: `string` - Large font size (1.125rem).
- `xl`: `string` - Extra large font size (1.25rem).

## Example

```typescript
import { FONT_SIZES } from 'your-sdk-name';

const smallFontSize = FONT_SIZES.sm;
console.log(smallFontSize); // Output: '0.875rem'
```
