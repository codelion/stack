# FONT_SIZES

An object that defines various font sizes used throughout the application.

## Properties

- `xs`: `string` - Extra small font size (0.75rem).
- `sm`: `string` - Small font size (0.875rem).
- `md`: `string` - Medium font size (1rem).
- `lg`: `string` - Large font size (1.125rem).
- `xl`: `string` - Extra large font size (1.25rem).

## Example

```typescript
import { FONT_SIZES } from '@stackframe/stack';

console.log(FONT_SIZES.sm); // Output: '0.875rem'
```
