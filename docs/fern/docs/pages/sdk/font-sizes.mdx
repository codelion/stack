# FONT_SIZES

A constant object containing predefined font sizes as key-value pairs. Each key represents a size name, and the corresponding value is the font size in rem units.

## Usage

```tsx
import { FONT_SIZES } from '@stackframe/stack';

const smallFontSize = FONT_SIZES.sm; // '0.875rem'
const largeFontSize = FONT_SIZES.lg; // '1.125rem'
```
