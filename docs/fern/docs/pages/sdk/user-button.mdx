# UserButton

The `UserButton` component renders a user button with a fallback skeleton display during loading. It takes various properties defined in the `UserButtonProps` interface.

## Parameters

- `props`: `UserButtonProps` - An object containing properties for customizing the `UserButton` component. Refer to `UserButtonProps` for detailed property definitions.

## Example

```tsx
import { UserButton } from '@stackframe/stack';

const props = {
  // Define your UserButtonProps here
};

<UserButton {...props} />
```
