# UserButton

Renders a button component for user interactions, with a fallback skeleton for loading states.

## Parameters

- `props`: `UserButtonProps` - The properties to customize the `UserButton` component.

## Example

```tsx
import { UserButton } from '@stackframe/stack';

const userButtonProps = {
  // Define your UserButtonProps here
};

<UserButton {...userButtonProps} />
```
