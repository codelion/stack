# UserButton

Renders a button for user interactions, displaying a loading state until the user information is fetched.

## Parameters

- `props`: `UserButtonProps` - The properties to configure the UserButton component, including user-related data.

## Example

```typescript
import { UserButton } from 'your-sdk-name';

function App() {
  return <UserButton userId={123} />;
}
```
