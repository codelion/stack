# UserButton

Renders a user button, showing a loading skeleton while user data is being fetched.

## Parameters

- `props`: `UserButtonProps` - Properties to configure the user button, including user information and any additional settings.

## Example

```typescript
import { UserButton } from '@stackframe/stack';

const App = () => {
  return <UserButton userId="123" />;
};
```
