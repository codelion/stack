# SECONDARY_FONT_COLORS

A constant object that defines secondary font colors for different themes.

## Properties

- `dark`: `string` - The secondary font color for dark themes. Value: `#a8a8a8`.
- `light`: `string` - The secondary font color for light themes. Value: `#737373`.

## Example

```tsx
import { SECONDARY_FONT_COLORS } from '@stackframe/stack';

const darkThemeColor = SECONDARY_FONT_COLORS.dark;
// darkThemeColor is '#a8a8a8'

const lightThemeColor = SECONDARY_FONT_COLORS.light;
// lightThemeColor is '#737373'
```
