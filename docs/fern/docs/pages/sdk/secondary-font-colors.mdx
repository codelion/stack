# SECONDARY_FONT_COLORS

An object that contains the secondary font colors for different themes.

## Properties

- `dark`: `string` - The secondary font color used in dark themes. Its value is `#a8a8a8`.
- `light`: `string` - The secondary font color used in light themes. Its value is `#737373`.

## Example

```typescript
import { SECONDARY_FONT_COLORS } from 'your-sdk-name';

console.log(SECONDARY_FONT_COLORS.dark); // Output: #a8a8a8
console.log(SECONDARY_FONT_COLORS.light); // Output: #737373
```
