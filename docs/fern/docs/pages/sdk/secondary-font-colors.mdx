# SECONDARY_FONT_COLORS

A constant object containing secondary font color values for dark and light themes.

## Properties

- `dark`: `string` - The secondary font color for dark theme ('#a8a8a8').
- `light`: `string` - The secondary font color for light theme ('#737373').

## Example

```tsx
import { SECONDARY_FONT_COLORS } from '@stackframe/stack';

const darkSecondaryColor = SECONDARY_FONT_COLORS.dark;
// darkSecondaryColor is '#a8a8a8'

const lightSecondaryColor = SECONDARY_FONT_COLORS.light;
// lightSecondaryColor is '#737373'
