# SECONDARY_FONT_COLORS

Contains secondary font colors for different themes.

## Properties

- `dark`: `string` - The font color used in dark theme. Value: `#a8a8a8`.
- `light`: `string` - The font color used in light theme. Value: `#737373`.

## Example

```typescript
import { SECONDARY_FONT_COLORS } from '@stackframe/stack';

const darkFontColor = SECONDARY_FONT_COLORS.dark;
const lightFontColor = SECONDARY_FONT_COLORS.light;

console.log(darkFontColor); // Output: #a8a8a8
console.log(lightFontColor); // Output: #737373
```
