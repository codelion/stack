# TeamSettings

Displays the settings for a team, including profile, members, and team information.

## Parameters

- `props`: `{ fullPage?: boolean, teamId: string }` - The properties for the `TeamSettings` component.
  - `fullPage`: `boolean` (optional) - If true, the component will be displayed as a full page.
  - `teamId`: `string` - The unique identifier for the team.

## Example

```typescript
import { TeamSettings } from 'your-sdk-name';

<TeamSettings teamId="12345" fullPage={true} />
```
