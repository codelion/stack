# SignOut

Renders a sign-out component that displays a predefined message card. Optionally, the message card can be displayed as a full page.

## Parameters

- `props`: `object` - An object containing the following properties:
  - `fullPage` (optional): `boolean` - If `true`, the message card is displayed as a full page. Defaults to `false`.

## Example

```tsx
import { SignOut } from '@stackframe/stack';

// Basic usage
<SignOut />

// Full page usage
<SignOut fullPage={true} />
```
