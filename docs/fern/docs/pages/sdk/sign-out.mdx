# SignOut

Signs out the currently authenticated user and displays a predefined 'signed out' message.

## Parameters

- `props`: `object`
  - `fullPage` (optional): `boolean` - If `true`, the message card will be displayed in full-page mode. Defaults to `false`.

## Example

```tsx
import { SignOut } from '@stackframe/stack';

// Basic usage
<SignOut />

// With fullPage as true
<SignOut fullPage={true} />
```

This component will automatically sign the user out upon rendering and show a message indicating the user has been signed out. The `fullPage` prop can be used to display the message in a full-page mode.
