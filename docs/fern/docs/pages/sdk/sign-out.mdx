# SignOut

Handles the user sign-out process and displays a message confirming the sign-out.

## Parameters

- `props`: `object` - The properties for the sign-out component.
  - `fullPage`: `boolean` (optional) - A flag indicating whether to display the message in full page mode.

## Returns

`JSX.Element` - A component that displays a sign-out message.

## Example

```typescript
import { SignOut } from 'your-sdk-name';

const App = () => (
  <div>
    <SignOut fullPage={true} />
  </div>
);

export default App;
```
