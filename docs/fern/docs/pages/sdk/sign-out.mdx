# SignOut

Signs out the current user and displays a message indicating the user has signed out.

## Parameters

- `props`: `{ fullPage?: boolean }` - An object containing props for the component.
  - `fullPage`: `boolean` (optional) - If true, the sign-out message will occupy the full page.

## Returns

`JSX.Element` - A component that renders a predefined message card indicating the user has signed out.

## Example

```typescript
import { SignOut } from '@stackframe/stack';

function App() {
  return <SignOut fullPage={true} />;
}
```
