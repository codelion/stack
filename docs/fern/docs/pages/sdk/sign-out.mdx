# SignOut

A React component that signs out the current user and displays a predefined sign-out message. It can be rendered as a full page or inline component.

## Parameters

- `fullPage` (optional): `boolean` - Determines whether the sign-out message should be displayed as a full page. If not provided, defaults to inline display.

## Example

```tsx
import { SignOut } from '@stackframe/stack';

// Inline sign-out component
function MyComponent() {
  return <SignOut />;
}

// Full-page sign-out component
function MyFullPageSignOut() {
  return <SignOut fullPage={true} />;
}
```

This component automatically handles the sign-out process for the current user and displays an appropriate message.}
```
