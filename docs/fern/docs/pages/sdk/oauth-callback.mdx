# OAuthCallback

Handles the OAuth callback process, including error handling and redirection. This component is used to manage the OAuth callback flow in your application.

## Parameters

- `props`: `object`
  - `fullPage` (optional): `boolean` - If true, the message card will take up the full page. Default is false.

## Example

```tsx
import { OAuthCallback } from '@stackframe/stack';

// Basic usage
<OAuthCallback />

// With fullPage option
<OAuthCallback fullPage={true} />
```
