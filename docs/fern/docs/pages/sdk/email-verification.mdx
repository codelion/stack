# EmailVerification

Renders an email verification component based on the provided verification code and page settings. Returns a JSX element indicating the verification status.

## Parameters

- `searchParams` (optional): `Record<string, string>` - An object containing search parameters. The `code` parameter is used for email verification. Defaults to an empty string if not provided.
- `fullPage` (optional): `boolean` - A boolean indicating whether the component should be rendered as a full page. Defaults to `false`.

## Example

```tsx
import { EmailVerification } from '@stackframe/stack';

// Example with default parameters
const verificationComponent = <EmailVerification />;

// Example with custom searchParams and fullPage
const customVerificationComponent = <EmailVerification searchParams={{ code: '123456' }} fullPage={true} />;
```
