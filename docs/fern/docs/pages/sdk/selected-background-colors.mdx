# selectedBackgroundColors

A constant object that holds the selected background colors for different themes.

## Properties

- `dark`: `string` - The background color for the dark theme, represented in RGBA format.
- `light`: `string` - The background color for the light theme, represented in RGBA format.

## Example

```typescript
import { SELECTED_BACKGROUND_COLORS } from '@stackframe/stack';

console.log(SELECTED_BACKGROUND_COLORS.dark); // Output: rgba(255, 255, 255, 0.1)
console.log(SELECTED_BACKGROUND_COLORS.light); // Output: rgba(0, 0, 0, 0.04)
```
