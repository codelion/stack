# SELECTED_BACKGROUND_COLORS

A constant object that defines background colors for selected elements in dark and light themes.

## Properties

- `dark`: `string` - The background color for selected elements in dark theme.
- `light`: `string` - The background color for selected elements in light theme.

## Example

```tsx
import { SELECTED_BACKGROUND_COLORS } from '@stackframe/stack';

const darkModeSelectedColor = SELECTED_BACKGROUND_COLORS.dark;
// darkModeSelectedColor is 'rgba(255, 255, 255, 0.1)'

const lightModeSelectedColor = SELECTED_BACKGROUND_COLORS.light;
// lightModeSelectedColor is 'rgba(0, 0, 0, 0.04)'
