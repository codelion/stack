# Project

Represents a project with its unique identifier, display name, and configuration settings.

## Properties

- `id`: `string` - A unique identifier for the project.
- `displayName`: `string` - The display name of the project.
- `config`: `ProjectConfig` - The configuration settings for the project.

## Example

```typescript
import { Project } from 'your-sdk-name';

const myProject: Project = {
  id: '12345',
  displayName: 'My Awesome Project',
  config: {
    /* configuration settings */
  }
};
```
