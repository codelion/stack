# StackServerApp

Type definition for the server-side application object in the Stack framework.

## Props

- **`createTeam`**: `(data: ServerTeamCreateOptions) => Promise<ServerTeam>`  
  Creates a new team.

- **`getServerUser`**: `() => Promise<ProjectCurrentServerUser<ProjectId> | null>`  
  Deprecated. Use `getUser()` instead.

- **`createUser`**: `(options: ServerUserCreateOptions) => Promise<ServerUser>`  
  Creates a new user.

- **`useUser`**:
  - `(options: GetUserOptions<HasTokenStore> & { or: 'redirect' }) => ProjectCurrentServerUser<ProjectId>`
  - `(options: GetUserOptions<HasTokenStore> & { or: 'throw' }) => ProjectCurrentServerUser<ProjectId>`
  - `(options?: GetUserOptions<HasTokenStore>) => ProjectCurrentServerUser<ProjectId> | null`

- **`getUser`**:
  - `(options: GetUserOptions<HasTokenStore> & { or: 'redirect' }) => Promise<ProjectCurrentServerUser<ProjectId>>`
  - `(options: GetUserOptions<HasTokenStore> & { or: 'throw' }) => Promise<ProjectCurrentServerUser<ProjectId>>`
  - `(options?: GetUserOptions<HasTokenStore>) => Promise<ProjectCurrentServerUser<ProjectId> | null>`

- **`users`**: AsyncStoreProperty for managing users.

- **`team`**: AsyncStoreProperty for managing a single team.

- **`teams`**: AsyncStoreProperty for managing multiple teams.
