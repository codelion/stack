# getCookie

Retrieves the value of a cookie by its name.

## Parameters

- `name`: `string` - The name of the cookie to retrieve.

## Returns

`string | null` - The value of the cookie if it exists, or `null` if the cookie is not found.

## Example

```typescript
import { getCookie } from '@stackframe/stack';

const userSession = getCookie('sessionId');
console.log(userSession); // Output: The value of the 'sessionId' cookie or null if not found.
```
