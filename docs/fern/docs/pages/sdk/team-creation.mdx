# TeamCreation

Creates a form for team creation in a project.

## Parameters

- `props`: `{ fullPage?: boolean }` - Optional properties for the component. The `fullPage` property determines whether the component should take the full page height.

## Example

```typescript
import { TeamCreation } from 'your-sdk-name';

const App = () => {
  return (
    <TeamCreation fullPage={true} />
  );
};
```
