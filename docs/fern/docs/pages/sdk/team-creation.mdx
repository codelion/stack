# TeamCreation

The `TeamCreation` component allows users to create a new team. It includes a form for entering the team's display name and handles the submission process.

## Parameters

- `props`: `object` - The properties object.
  - `fullPage` (optional): `boolean` - If true, the component will be displayed as a full page. Defaults to `false`.

## Example

```tsx
import { TeamCreation } from '@stackframe/stack';

// Basic usage
<TeamCreation />

// Full page usage
<TeamCreation fullPage={true} />
```
