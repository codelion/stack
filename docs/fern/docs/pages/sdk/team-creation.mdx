# TeamCreation

A component to create a new team in the application.

## Parameters

- `props`: `{ fullPage?: boolean }` - Props to customize the component. The `fullPage` property determines if the component should take the full page view.

## Example

```typescript
import { TeamCreation } from '@stackframe/stack';

const App = () => {
  return <TeamCreation fullPage={true} />;
};
```
