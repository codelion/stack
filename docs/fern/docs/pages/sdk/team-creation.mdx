# TeamCreation

The `TeamCreation` function renders a form that allows users to create a new team.

## Parameters

- `props`: `object`
  - `fullPage` (optional): `boolean` - Determines whether the form should be displayed as a full page. Default is `false`.

## Example

```tsx
import { TeamCreation } from '@stackframe/stack';

// Full page form
<TeamCreation fullPage={true} />

// Default form
<TeamCreation />
```
