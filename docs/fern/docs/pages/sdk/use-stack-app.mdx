# useStackApp

A hook to access the Stack client application within a React component.

## Parameters

- `options`: `{ projectIdMustMatch?: ProjectId }` (optional) - An object containing options for the hook.
  - `projectIdMustMatch`: `ProjectId` - An optional project ID that must match the current stack application's project ID.

## Returns

`StackClientApp<true, ProjectId>` - The current stack client application, cast to the appropriate type.

## Example

```typescript
import { useStackApp } from '@stackframe/stack';

function MyComponent() {
  const app = useStackApp({ projectIdMustMatch: 'myProjectId' });
  console.log(app);
  return <div>{app.name}</div>;
}
```
