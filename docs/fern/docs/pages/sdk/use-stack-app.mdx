# useStackApp

A hook to access the StackClientApp within a StackProvider context.

## Parameters

- `options`: `object` (optional) - An object to configure the hook.
  - `projectIdMustMatch`: `ProjectId` (optional) - A project ID that must match the project ID of the StackClientApp.

## Returns

`StackClientApp<true, ProjectId>` - The StackClientApp instance associated with the current StackProvider context.

## Example

```typescript
import { useStackApp } from 'your-sdk-name';

const MyComponent = () => {
  const stackApp = useStackApp({ projectIdMustMatch: 'my-project-id' });
  
  // Use stackApp as needed
};
```
