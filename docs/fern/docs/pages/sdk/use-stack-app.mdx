The `StackServerApp` object allows you to interact with the Stack API. You can access it using the `stack` prop provided by the `withStackApp` higher-order component.

Example:

```jsx
import { withStackApp } from "@stackframe/stack";

function MyComponent({ stack }) {
  // Access StackServerApp properties and methods
  const signInUrl = stack.getSignInUrl();
  return <div>Sign In URL: {signInUrl}</div>;
}

export default withStackApp(MyComponent);
```

For more information about the `StackServerApp` object, refer to the [App](./stack-app.mdx) documentation.
