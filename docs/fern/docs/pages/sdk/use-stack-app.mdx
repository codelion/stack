The `useStackServerApp` hook returns the `StackServerApp` object that you can use to interact with the Stack API. If you want to learn more about the `StackServerApp` object, check out the [App](./stack-app.mdx) documentation.

Example:

```jsx
import { useStackServerApp } from "@stackframe/stack";

function MyComponent() {
  const stackApp = useStackServerApp();
  return <div>Sign In URL: {stackApp.urls.signIn}</div>;
}
```
