# useStackApp

The `useStackApp` hook provides access to the `StackClientApp` within a `StackProvider` context. It ensures that the project ID matches the expected value if specified.

## Parameters

- `options` (optional): `object` - Configuration options.
  - `projectIdMustMatch` (optional): `ProjectId` - If provided, the project ID must match this value.

## Returns

- `StackClientApp<true, ProjectId>` - The stack application client.

## Example

```tsx
import { useStackApp } from '@stackframe/stack';
import { StackProvider } from '@stackframe/stack';

const MyComponent = () => {
  const stackApp = useStackApp({ projectIdMustMatch: 'my-project-id' });
  return <div>Project ID: {stackApp.projectId}</div>;
};

const App = () => (
  <StackProvider>
    <MyComponent />
  </StackProvider>
);
```
