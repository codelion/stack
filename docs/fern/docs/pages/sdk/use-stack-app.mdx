# useStackApp

Returns the StackServerApp instance for the current project. Must be used within a StackProvider.

## Parameters

- `options` (optional): An object with the following properties:
  - `projectIdMustMatch` (optional): `ProjectId` - If provided, ensures the returned app matches this project ID.

## Example

```tsx
import { useStackApp } from '@stackframe/stack';

function MyComponent() {
  // Basic usage
  const app = useStackApp();

  // With project ID check
  const app = useStackApp({ projectIdMustMatch: 'my-project-id' });

  // Using the app
  const user = app.useUser();
  const project = app.useProject();

  // Redirect example
  const handleSignOut = () => {
    app.redirectToSignOut();
  };

  // ... rest of your component
}
```
