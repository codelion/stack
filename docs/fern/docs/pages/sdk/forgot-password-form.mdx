# ForgotPasswordForm

A form component for sending a forgot password email. It uses the `useForm` hook for form handling and validation.

## Parameters

- `onSent` (optional): `() => void` - A callback function that is called after the forgot password email is successfully sent.

## Example

```tsx
import { ForgotPasswordForm } from '@stackframe/stack';

function App() {
  const handleSent = () => {
    alert('Password reset email sent!');
  };

  return (
    <div>
      <h1>Forgot Password</h1>
      <ForgotPasswordForm onSent={handleSent} />
    </div>
  );
}

export default App;
```
