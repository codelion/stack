# ForgotPasswordForm

Renders a form to request a password reset email. This form uses the `useForm` hook for handling form state and validation, and sends the request via the `stackApp` instance.

## Parameters

- `onSent` (optional): `() => void` - A callback function to be executed once the password reset email is sent. Defaults to no action if not provided.

## Example

```tsx
import { ForgotPasswordForm } from '@stackframe/stack';

const handleSent = () => {
  alert('Password reset email sent!');
};

const App = () => (
  <ForgotPasswordForm onSent={handleSent} />
);

// Without optional parameter
const AppWithoutCallback = () => (
  <ForgotPasswordForm />
);
```
