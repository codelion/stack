# ForgotPasswordForm

A React component for a forgot password form that sends an email to reset the password.

## Parameters

- `onSent`: `() => void` (optional) - A callback function that is called when the email has been successfully sent.

## Example

```typescript
import { ForgotPasswordForm } from '@stackframe/stack';

const handleSent = () => {
  console.log('Password reset email sent!');
};

<ForgotPasswordForm onSent={handleSent} />
```
