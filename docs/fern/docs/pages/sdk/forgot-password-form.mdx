# ForgotPasswordForm

A React component that renders a form for users to request a password reset email. It handles form submission, validation, and displays loading state.

## Parameters

- `onSent` (optional): `() => void` - A callback function that is called after the forgot password email is successfully sent.

## Example

```tsx
import { ForgotPasswordForm } from '@stackframe/stack';

function PasswordResetPage() {
  const handleEmailSent = () => {
    // Handle successful email sent
    console.log('Password reset email sent successfully');
  };

  return (
    <div>
      <h1>Forgot Password</h1>
      <ForgotPasswordForm onSent={handleEmailSent} />
    </div>
  );
}
```

In this example, the `ForgotPasswordForm` is used within a `PasswordResetPage` component. The `onSent` callback is provided to handle the successful email sent event.}

// Without onSent callback
function SimplePasswordResetPage() {
  return (
