# ForgotPasswordForm

A form component for sending a forgot password email.

## Parameters

- `onSent`: `() => void` (optional) - A callback function that is called when the email has been sent successfully.

## Example

```typescript
import { ForgotPasswordForm } from 'your-sdk-name';

const handleSent = () => {
  console.log('Email sent successfully!');
};

<ForgotPasswordForm onSent={handleSent} />
```
