# ForgotPassword

A React component that manages the forgot password flow for users.

## Parameters

- `fullPage`: `boolean` (optional) - A flag indicating whether the component should take up the full page layout. Defaults to `false`.

## Example

```typescript
import { ForgotPassword } from 'your-sdk-name';

const App = () => (
  <ForgotPassword fullPage={true} />
);
```
