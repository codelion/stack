# ForgotPassword

Renders a password reset component with full page or partial page views, depending on the `fullPage` parameter. Displays different messages based on the user's state (signed in, email sent, or not signed in).

## Parameters

- `fullPage` (optional): `boolean` - Determines if the component should render as a full page. Default is `false`.

## Example

```tsx
import { ForgotPassword } from '@stackframe/stack';

// Render partial page view
<ForgotPassword />;

// Render full page view
<ForgotPassword fullPage={true} />;
```
