# ForgotPassword

Renders a password reset form. If the user is already signed in, it shows a message indicating that. If the reset email has been sent, it shows a confirmation message.

## Parameters

- `fullPage` (optional): `boolean` - Determines if the component should take up the full page. Defaults to `false`.

## Example

```tsx
import { ForgotPassword } from '@stackframe/stack';

// Default usage
<ForgotPassword />

// Full page usage
<ForgotPassword fullPage={true} />
```
