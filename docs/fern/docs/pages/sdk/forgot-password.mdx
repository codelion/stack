# ForgotPassword

A React component that renders a forgot password interface. It displays different content based on the user's authentication status and whether a password reset email has been sent.

## Parameters

- `fullPage` (optional): `boolean` - Determines whether the component should be rendered in full-page mode. Default is `false`.

## Returns

Returns a React component that displays either a sign-in message, an email sent confirmation, or a password reset form based on the current state.

## Example

```tsx
import { ForgotPassword } from '@stackframe/stack';

// Basic usage
function PasswordResetPage() {
  return <ForgotPassword />;
}

// Full-page mode
function FullPagePasswordResetPage() {
  return <ForgotPassword fullPage={true} />;
}
```
