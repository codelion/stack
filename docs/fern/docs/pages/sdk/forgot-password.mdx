# ForgotPassword

Renders a password reset page with a form for users to reset their password. It shows different messages based on the user's authentication status and whether the reset email has been sent.

## Parameters

- `fullPage`: `boolean` (optional) - Determines whether the component is displayed in full page mode. Defaults to `false`.

## Example

```typescript
import { ForgotPassword } from '@stackframe/stack';

<ForgotPassword fullPage={true} />
```
