# membersSettings

Renders a component for managing team members, allowing inviting and listing existing members based on user permissions.

## Parameters

- `props`: `{ team: Team }` - An object containing the `team` for which members are to be managed.

## Example

```typescript
import { membersSettings } from '@stackframe/stack';

const team = {/* team object */};

<membersSettings team={team} />;
```
