# FormWarningText

Displays a warning message if provided.

## Parameters

- `text`: `string` (optional) - The warning message to display. If no text is provided, nothing will be rendered.

## Returns

`JSX.Element | null` - Returns a `div` element with the warning text styled in red, or `null` if no text is provided.

## Example

```typescript
import { FormWarningText } from 'your-sdk-name';

<FormWarningText text="This field is required." />
```
