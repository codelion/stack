# TeamInvitation

Renders a team invitation component based on the provided search parameters. It handles different states such as invalid, expired, or used invitation links.

## Parameters

- `fullPage` (optional): `boolean` - Determines if the component should be rendered in full-page mode. Defaults to `false`.
- `searchParams`: `Record<string, string>` - An object containing search parameters, including the invitation code.

## Example

```tsx
import { TeamInvitation } from '@stackframe/stack';

const searchParams = { code: 'INVITE123' };

// Basic usage
<TeamInvitation searchParams={searchParams} />;

// Full-page mode
<TeamInvitation fullPage={true} searchParams={searchParams} />;
```
