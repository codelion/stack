# FONT_FAMILY

Represents the default font family used in the application.

## Value

- `string` - A comma-separated list of font families including system defaults and emoji fonts.

## Example

```typescript
import { FONT_FAMILY } from 'your-sdk-name';

console.log(FONT_FAMILY); // Output: 'ui-sans-serif, system-ui, sans-serif,
