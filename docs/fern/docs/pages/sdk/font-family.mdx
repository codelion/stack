# FONT_FAMILY

A constant string representing a list of font families to be used in the application. This string can be used in CSS or inline styles to set the font family for elements.

## Example

```tsx
import { FONT_FAMILY } from '@stackframe/stack';

const styles = {
  fontFamily: FONT_FAMILY
};

// Usage in a React component
const MyComponent = () => <div style={styles}>Hello, World!</div>;
```
