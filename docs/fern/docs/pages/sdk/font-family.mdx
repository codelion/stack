# FONT_FAMILY

A constant string that defines a fallback-rich font family stack for use in user interfaces.

## Example

```tsx
import { FONT_FAMILY } from '@stackframe/stack';

const styles = {
  fontFamily: FONT_FAMILY
};
``````tsx
import { FONT_FAMILY } from '@stackframe/stack';

  fontFamily: FONT_FAMILY
