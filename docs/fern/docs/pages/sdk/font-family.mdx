# FONT_FAMILY

`FONT_FAMILY` is a constant providing a comma-separated string of font families. It is suitable for use in CSS to specify a font stack for styling text.

## Example

```tsx
import { FONT_FAMILY } from '@stackframe/stack';

const styles = {
  fontFamily: FONT_FAMILY
};

// Applying the styles to a React component
const MyComponent = () => <div style={styles}>Hello, world!</div>;
```
