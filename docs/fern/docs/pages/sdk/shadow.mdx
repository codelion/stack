# SHADOW

A constant string representing a subtle box shadow CSS value. This shadow provides a light elevation effect to elements.

## Value

`'0 1px 2px 0 rgba(0, 0, 0, 0.05)'`

## Example

```tsx
import { SHADOW } from '@stackframe/stack';

const CardComponent = () => (
  <div style={{ boxShadow: SHADOW }}>
    {/* Card content */}
  </div>
);
```  boxShadow: SHADOW
};
