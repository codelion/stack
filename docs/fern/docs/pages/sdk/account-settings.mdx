# AccountSettings

Renders the account settings interface with sections for profile, security, and other settings.

## Parameters

- `fullPage`: `boolean` (optional) - If `true`, the component will render in full page width.

## Returns

`JSX.Element` - A rendered component displaying the account settings sections.

## Example

```typescript
import { AccountSettings } from '@stackframe/stack';

function App() {
  return <AccountSettings fullPage={true} />;
}
```
