# AccountSettings

The `AccountSettings` component renders user account settings, including profile, security, and other settings. It can be displayed as a full page or within a sidebar layout.

## Parameters

- `fullPage` (optional): `boolean` - Determines if the settings should be displayed as a full page. Defaults to `false`.

## Example

```tsx
import { AccountSettings } from '@stackframe/stack';

// Render as a sidebar layout
<AccountSettings />

// Render as a full page
<AccountSettings fullPage={true} />
```
