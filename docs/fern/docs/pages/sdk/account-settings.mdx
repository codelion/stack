# AccountSettings

Renders an account settings component with different sections such as profile, security, and general settings. The component can be displayed as a full page or embedded within another layout.

## Parameters

- `fullPage` (optional): `boolean` - Determines whether to render the component as a full page. Defaults to `false`.

## Returns

Returns a React component that displays the account settings UI.

## Example

```tsx
import { AccountSettings } from '@stackframe/stack';

// Render as an embedded component
function MyComponent() {
  return (
    <div>
      <h1>My App</h1>
      <AccountSettings />
    </div>
  );
}

// Render as a full page
function FullPageSettings() {
  return <AccountSettings fullPage={true} />;
}
```
