# AccountSettings

Renders account settings for a user, allowing for a full-page or sidebar layout.

## Parameters

- `fullPage`: `boolean` (optional) - Determines whether the settings should be displayed in a full-page layout. Defaults to `false`.

## Returns

`JSX.Element` - A rendered component representing the user's account settings.

## Example

```typescript
import { AccountSettings } from 'your-sdk-name';

const App = () => {
  return <AccountSettings fullPage={true} />;
};
```
