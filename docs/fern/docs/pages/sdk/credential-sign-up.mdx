# CredentialSignUp

Renders a sign-up form component with email and password fields. Handles form submission and validation.

## Parameters

This component does not accept any parameters.

## Example

```tsx
import { CredentialSignUp } from '@stackframe/stack';

function SignUpPage() {
  return (
    <div>
      <h1>Sign Up</h1>
      <CredentialSignUp />
    </div>
  );
}
```
