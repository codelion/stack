# CredentialSignUp

The `CredentialSignUp` component provides a form for user sign-up using email and password. It handles form validation, submission, and error display.

## Usage

```tsx
import { CredentialSignUp } from '@stackframe/stack';

function App() {
  return <CredentialSignUp />;
}
```

## Description

- **register**: Registers the input fields for form validation.
- **handleSubmit**: Handles form submission.
- **setError**: Sets error messages for form fields.
- **formState.errors**: Contains validation errors.
- **clearErrors**: Clears validation errors.
- **useForm**: Hook from `react-hook-form` for form handling.
- **yupResolver**: Resolver for form validation using Yup schema.
- **useStackApp**: Custom hook to access the Stack application context.
- **useState**: React hook to manage loading state.

## Example

```tsx
import { CredentialSignUp } from '@stackframe/stack';

function App() {
  return <CredentialSignUp />;
}
```

This component renders a sign-up form with fields for email, password, and password confirmation. It validates the input and displays error messages if the validation fails. Upon successful submission, it attempts to sign up the user using the provided credentials.
