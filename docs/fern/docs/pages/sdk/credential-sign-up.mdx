# CredentialSignUp

Renders a sign-up form for user credentials, including email and password.

## Parameters

This component does not accept props. It maintains its own internal state.

## Example

```typescript
import { CredentialSignUp } from '@stackframe/stack';

const App = () => {
  return <CredentialSignUp />;
};

export default App;
```
