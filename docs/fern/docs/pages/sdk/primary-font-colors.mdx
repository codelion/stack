# PRIMARY_FONT_COLORS

Provides the primary font colors for different themes.

## Properties

- `dark`: `string` - The primary font color for dark themes. Set to `'white'`.
- `light`: `string` - The primary font color for light themes. Set to `'black'`.

## Example

```tsx
import { PRIMARY_FONT_COLORS } from '@stackframe/stack';

const darkModeFontColor = PRIMARY_FONT_COLORS.dark; // 'white'
const lightModeFontColor = PRIMARY_FONT_COLORS.light; // 'black'
```
