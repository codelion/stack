# PRIMARY_FONT_COLORS

Stores the primary font colors for different themes.

## Properties

- `dark`: `string` - The color for dark theme, set to `white`.
- `light`: `string` - The color for light theme, set to `black`.

## Example

```typescript
import { PRIMARY_FONT_COLORS } from '@stackframe/stack';

console.log(PRIMARY_FONT_COLORS.dark); // Output: 'white'
console.log(PRIMARY_FONT_COLORS.light); // Output: 'black'
```
