# PRIMARY_FONT_COLORS

A constant object that defines the primary font colors for dark and light themes.

## Properties

- `dark`: `'white'` - The primary font color for dark theme.
- `light`: `'black'` - The primary font color for light theme.

## Example

```tsx
import { PRIMARY_FONT_COLORS } from '@stackframe/stack';

const darkModeColor = PRIMARY_FONT_COLORS.dark;
// darkModeColor is 'white'

const lightModeColor = PRIMARY_FONT_COLORS.light;
// lightModeColor is 'black'
```// Using the primary font colors
const darkModeColor = PRIMARY_FONT_COLORS.dark;
// darkModeColor is 'white'

const lightModeColor = PRIMARY_FONT_COLORS.light;
// lightModeColor is 'black'
```
