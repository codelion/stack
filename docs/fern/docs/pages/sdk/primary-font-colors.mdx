# PRIMARY_FONT_COLORS

An object that defines the primary font colors for different themes.

## Properties

- `dark`: `string` - The font color for the dark theme, set to 'white'.
- `light`: `string` - The font color for the light theme, set to 'black'.

## Example

```typescript
import { PRIMARY_FONT_COLORS } from 'your-sdk-name';

console.log(PRIMARY_FONT_COLORS.dark); // Output: "white"
console.log(PRIMARY_FONT_COLORS.light); // Output: "black"
```
