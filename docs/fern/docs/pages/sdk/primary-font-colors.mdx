# PRIMARY_FONT_COLORS

A constant object that defines primary font colors for different themes.

## Properties

- `dark`: `string` - The font color for dark themes. Value is `'white'`.
- `light`: `string` - The font color for light themes. Value is `'black'`.

## Example

```tsx
import { PRIMARY_FONT_COLORS } from '@stackframe/stack';

const darkThemeColor = PRIMARY_FONT_COLORS.dark;
// darkThemeColor is 'white'

const lightThemeColor = PRIMARY_FONT_COLORS.light;
// lightThemeColor is 'black'
```
