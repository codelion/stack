# processUser

Processes a user object and logs the user's name to the console.

## Parameters

- `user`: `User` - An object representing the user. The user object must have a `name` property which is a string.

## Example

```tsx
import { processUser } from '@stackframe/stack';

const user = {
  name: 'John Doe'
};

processUser(user);
// Logs: 'John Doe'
```
