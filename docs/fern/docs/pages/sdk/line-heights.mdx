# LINE_HEIGHTS

A constant object that defines standard line heights for different sizes. Each size is associated with a specific line height value.

## Properties

- `xs`: `1rem` - Extra small line height.
- `sm`: `1.25rem` - Small line height.
- `md`: `1.5rem` - Medium line height.
- `lg`: `1.75rem` - Large line height.
- `xl`: `2rem` - Extra large line height.

## Example

```tsx
import { LINE_HEIGHTS } from '@stackframe/stack';

const smallLineHeight = LINE_HEIGHTS.sm;
// smallLineHeight is '1.25rem'

const largeLineHeight = LINE_HEIGHTS.lg;
// largeLineHeight is '1.75rem'
```
