# LINE_HEIGHTS

A constant object that defines standard line heights for different text sizes.

## Properties

- `xs`: `string` - The line height for extra small text, set to `1rem`.
- `sm`: `string` - The line height for small text, set to `1.25rem`.
- `md`: `string` - The line height for medium text, set to `1.5rem`.
- `lg`: `string` - The line height for large text, set to `1.75rem`.
- `xl`: `string` - The line height for extra large text, set to `2rem`.

## Example

```typescript
import { LINE_HEIGHTS } from 'your-sdk-name';

console.log(LINE_HEIGHTS.md); // Output: '1.5rem'
```
