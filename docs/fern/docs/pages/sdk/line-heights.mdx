# LINE_HEIGHTS

A constant object that defines various line heights for use in styling.

## Properties

- `xs`: `string` - Line height for extra small text, set to `1rem`.
- `sm`: `string` - Line height for small text, set to `1.25rem`.
- `md`: `string` - Line height for medium text, set to `1.5rem`.
- `lg`: `string` - Line height for large text, set to `1.75rem`.
- `xl`: `string` - Line height for extra large text, set to `2rem`.

## Example

```typescript
import { LINE_HEIGHTS } from '@stackframe/stack';

console.log(LINE_HEIGHTS.xs); // Output: '1rem'
console.log(LINE_HEIGHTS.md); // Output: '1.5rem'
```- `lg`: `string` - Line height for large text, set to `1.75rem`.
- `xl`: `string` - Line height for extra large text, set to `2rem`.
