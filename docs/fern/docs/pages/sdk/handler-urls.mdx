# HandlerUrls

Defines a set of URLs used for various user actions in the application.

## Properties

- `handler`: `string` - The URL for handling requests.
- `signIn`: `string` - The URL for the sign-in page.
- `signUp`: `string` - The URL for the sign-up page.
- `afterSignIn`: `string` - The URL to redirect to after a successful sign-in.
- `afterSignUp`: `string` - The URL to redirect to after a successful sign-up.
- `signOut`: `string` - The URL for the sign-out action.
- `afterSignOut`: `string` - The URL to redirect to after a successful sign-out.
- `emailVerification`: `string` - The URL for email verification.
- `passwordReset`: `string` - The URL for password reset requests.
- `forgotPassword`: `string` - The URL for the forgot password page.
- `home`: `string` - The URL for the home page.
- `oauthCallback`: `string` - The URL for OAuth callbacks.
- `magicLinkCallback`: `string` - The URL for magic link callbacks.
- `accountSettings`: `string` - The URL for account settings.
- `teamInvitation`: `string` - The URL for team invitation actions.
- `teamCreation`: `string` - The URL for team creation actions.
- `error`: `string` - The URL for handling errors.
