# CredentialSignIn

Renders a sign-in form for users to enter their credentials and submit for authentication.

## Returns

`JSX.Element` - A React component rendering the sign-in form with email and password fields.

## Example

```typescript
import { CredentialSignIn } from '@stackframe/stack';

function App() {
  return <CredentialSignIn />;
}
```
