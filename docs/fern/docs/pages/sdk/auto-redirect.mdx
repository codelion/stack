# autoRedirect

Redirects the user to a specified URL if the `auto-redirect-url` parameter is present in the query string.

## Parameters

This function does not take any parameters.

## Returns

This function does not return a value. It redirects the browser to the specified URL if conditions are met.

## Example

```typescript
import { autoRedirect } from '@stackframe/stack';

autoRedirect();
```
