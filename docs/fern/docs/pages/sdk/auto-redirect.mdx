# autoRedirect

Automatically redirects the current page to a URL specified in the 'auto-redirect-url' search parameter, if present. Ensures the URL is of the same origin as the current page before redirecting.

## Example

```tsx
import { autoRedirect } from '@stackframe/stack';

// Assume the current URL is 'https://example.com?auto-redirect-url=https://example.com/redirect'

// This will automatically redirect to 'https://example.com/redirect'
autoRedirect();
```
