# autoRedirect

Redirects the user to a specified URL if the `auto-redirect-url` parameter is present in the current URL's query string.

## Parameters

This function does not take any parameters.

## Returns

This function does not return a value. It performs a redirect based on the presence of the `auto-redirect-url` query parameter.

## Example

```typescript
import { autoRedirect } from 'your-sdk-name';

autoRedirect(); // Call this function to handle automatic redirection based on the query parameter.
```
