# PasswordResetForm

A component for resetting a user's password.

## Parameters

- `code`: `string` - The password reset code.
- `fullPage`: `boolean` (optional) - Determines if the form should be displayed in full page mode. Defaults to `false`.

## Example

```typescript
import PasswordResetForm from 'your-sdk-name';

function App() {
  return <PasswordResetForm code='your-code-here' />;
}
```
