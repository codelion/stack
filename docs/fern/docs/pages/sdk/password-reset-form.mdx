# PasswordResetForm

A component for resetting a user's password.

## Parameters

- `code`: `string` - The password reset code to validate the request.
- `fullPage`: `boolean` (optional) - Determines if the form should occupy the full page. Defaults to `false`.

## Example

```typescript
import PasswordResetForm from '@stackframe/stack';

const App = () => {
  return <PasswordResetForm code='your-reset-code' fullPage={true} />;
};
```
