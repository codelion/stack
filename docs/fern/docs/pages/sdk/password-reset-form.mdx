# PasswordResetForm

A React component that renders a password reset form. It handles the password reset process and displays appropriate messages based on the result.

## Parameters

- `code`: `string` - The password reset code received from the reset password link.
- `fullPage` (optional): `boolean` - Determines if the form should be displayed in full-page mode. Default is `false`.

## Returns

Returns a React component that renders the password reset form or appropriate messages based on the reset process state.

## Example

```tsx
import { PasswordResetForm } from '@stackframe/stack';

// Basic usage
function ResetPasswordPage({ code }) {
  return <PasswordResetForm code={code} />;
}

// Full page mode
function FullPageResetPasswordPage({ code }) {
  return <PasswordResetForm code={code} fullPage={true} />;
}
```

## Component Behavior

- The component uses the `useForm` hook for form handling and validation.
- It displays input fields for the new password and password confirmation.
- The form submission is handled asynchronously, with loading state management.
- After a successful password reset, it displays a success message.
- If the reset fails, it shows an error message.
- The component can be rendered in full-page mode if the `fullPage` prop is set to `true`.
