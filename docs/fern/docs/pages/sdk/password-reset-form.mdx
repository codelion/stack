# PasswordResetForm

The `PasswordResetForm` component renders a form to reset a user's password using a provided reset code. It handles the password input and submission process, showing appropriate messages based on the outcome.

## Parameters

- `code`: `string` - The password reset code to be used for resetting the password.
- `fullPage` (optional): `boolean` - A flag indicating whether the form should be displayed as a full page. Defaults to `false`.

## Example

```tsx
import PasswordResetForm from '@stackframe/stack';

export default function App() {
  const resetCode = 'example-reset-code';

  return (
    <div>
      <PasswordResetForm code={resetCode} />
      <PasswordResetForm code={resetCode} fullPage={true} />
    </div>
  );
}
```
