# PasswordResetForm

A form component for resetting a user's password. It handles form submission, validation, and displays appropriate messages based on the reset status.

## Parameters

- `code`: `string` - The reset code required to reset the password.
- `fullPage` (optional): `boolean` - If `true`, the form will be displayed as a full page. Defaults to `false`.

## Example

```tsx
import PasswordResetForm from '@stackframe/stack';

// Usage with required parameter
<PasswordResetForm code='reset-code' />

// Usage with both parameters
<PasswordResetForm code='reset-code' fullPage={true} />
```
