# StackHandler

`StackHandler` is an asynchronous function used to manage authentication and account-related routes in a `@stackframe/stack` application. It handles different paths and redirects to appropriate components based on the provided `stack` parameter.

## Parameters

- `app`: `StackServerApp<HasTokenStore>` - The application instance.
- `params` (optional): `{ stack?: string[] }` - An object containing a `stack` property which is an array of route segments.
- `searchParams` (optional): `Record<string, string>` - An object representing the search parameters to be appended to the URL.
- `fullPage` (optional): `boolean` - A boolean to determine whether to render the full page layout. Defaults to `false`.

## Example

```tsx
import { StackServerApp } from '@stackframe/stack';
import StackHandler from '@stackframe/stack';

const app = new StackServerApp({ /* app configuration */ });

const params = { stack: ['sign-in'] };
const searchParams = { redirect: '/home' };
const fullPage = true;

const handlerOutput = await StackHandler({ app, params, searchParams, fullPage });

// Use handlerOutput in your component
```
