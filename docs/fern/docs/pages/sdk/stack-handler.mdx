# StackHandler

Handles various stack-related actions based on the provided parameters.

## Parameters

- `app`: `StackServerApp<HasTokenStore>` - The application instance which contains methods and properties related to the stack handling.
- `params`: `{ stack?: string[] }` (optional) - An array of stack names to process. Must be included for the handler to work.
- `searchParams`: `Record<string, string>` (optional) - A record of search parameters to include in the redirected URLs.
- `fullPage`: `boolean` (optional) - Determines if the response should be a full page rendering.

## Returns

`JSX.Element` - A rendered component based on the stack and available paths provided, or a message indicating an invalid setup.

## Example

```typescript
import { StackHandler } from '@stackframe/stack';

const app = getAppInstance(); // Assume this function provides the StackServerApp instance

<StackHandler app={app} params={{ stack: ['sign-in'] }} searchParams={{ redirectTo: '/home' }} fullPage={true} />
```
