# StackHandler

`StackHandler` is an asynchronous function that handles various stack-related routes and redirects within the application. It returns different components based on the provided stack path.

## Parameters

- `app`: `StackServerApp<HasTokenStore>` - The application instance.
- `params` (optional): `{ stack?: string[] }` - An object containing an optional `stack` array of strings. Default is `{}`.
- `searchParams` (optional): `Record<string, string>` - An object containing search parameters as key-value pairs. Default is `{}`.
- `fullPage` (optional): `boolean` - A boolean indicating whether the page should be displayed in full-page mode. Default is `false`.

## Example

```tsx
import StackHandler from '@stackframe/stack';

const app = getAppInstance(); // Assume this function returns a StackServerApp instance
const params = { stack: ['sign-in'] };
const searchParams = { ref: 'newsletter' };
const fullPage = true;

const result = await StackHandler({ app, params, searchParams, fullPage });
```

```tsx
import StackHandler from '@stackframe/stack';

const app = getAppInstance(); // Assume this function returns a StackServerApp instance
const params = { stack: ['team-settings', 'team-id'] };
```
