# StackHandler

Handles routing for various authentication and user management pages in a Stack application.

## Parameters

- `app`: `StackServerApp<HasTokenStore>` - The Stack server application instance.
- `params` (optional): `{ stack?: string[] }` - URL parameters, where `stack` is an array of path segments.
- `searchParams` (optional): `Record<string, string>` - Query parameters as key-value pairs.
- `fullPage` (optional): `boolean` - Whether to render the component as a full page.

## Example

```tsx
import { StackHandler } from '@stackframe/stack';

const App = ({ app }) => {
  return (
    <StackHandler
      app={app}
      params={{ stack: ['sign-in'] }}
      searchParams={{ redirect: '/dashboard' }}
      fullPage={true}
    />
  );
};
```
