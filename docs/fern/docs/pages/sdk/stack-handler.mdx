# StackHandler

A handler function for managing various authentication and user-related pages in a Stack application.

## Props

- `app`: `StackServerApp<HasTokenStore>` - The Stack server application instance.
- `params` (optional): `{ stack?: string[] }` - URL parameters, where `stack` is an array of path segments.
- `searchParams` (optional): `Record<string, string>` - Query parameters from the URL.
- `fullPage` (optional): `boolean` - Whether to render the component in full-page mode. Default is `false`.

## Example

```tsx
import StackHandler from '@stackframe/stack';

export default async function Page({
  params,
  searchParams
}: {
  params: { stack: string[] },
  searchParams: { [key: string]: string | string[] | undefined }
}) {
  return (
    <StackHandler
      app={yourStackServerApp}
      params={params}
      searchParams={searchParams}
      fullPage={true}
    />
  );
}
```
