# StackHandler

A React component that handles various authentication and user management routes for a Stack application. It renders different components based on the current path and application state.

## Parameters

- `app`: `StackServerApp<HasTokenStore>` - The Stack server application instance.
- `params` (optional): `{ stack?: string[] }` - An object containing the stack parameter, which is an array of path segments.
- `searchParams` (optional): `Record<string, string>` - An object containing query parameters.
- `fullPage` (optional): `boolean` - A flag to indicate if the component should render in full-page mode. Defaults to `false`.

## Example

```tsx
import { StackHandler } from '@stackframe/stack';

// Basic usage
const MyApp = ({ app }) => (
  <StackHandler app={app} />
);

// With optional parameters
const MyAppWithOptions = ({ app, stack, searchParams }) => (
  <StackHandler
    app={app}
    params={{ stack }}
    searchParams={searchParams}
    fullPage={true}
  />
);
```
