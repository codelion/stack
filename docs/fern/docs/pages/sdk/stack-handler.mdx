# StackHandler

Handles various stack-related routes in the application.

## Parameters

- `app`: `StackServerApp<HasTokenStore>` - The server application instance.
- `params`: `{ stack?: string[] }` (optional) - An array of stack identifiers. Must be provided to use this handler.
- `searchParams`: `Record<string, string>` (optional) - An object containing search parameters for the request.
- `fullPage`: `boolean` (optional) - Indicates whether to render a full page or a part of it.

## Returns

`JSX.Element` - Returns the appropriate component based on the stack route or a message card for invalid setups.

## Example

```typescript
import StackHandler from 'your-sdk-name';

const MyComponent = () => {
  return <StackHandler app={myApp} params={{ stack: ['signIn'] }} />;
};
```
